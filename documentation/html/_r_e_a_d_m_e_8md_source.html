3 ## Functionality:
4  * Create a Generic FlipBook using a custom library for book creation.
5  * Flipbook pages contain images or text defined by the User.
6  * Animation of Flipping of Pages with a specified Delay.
7  * We can render custom text, images onto the Pages.
8  * Navigation through the Pages.
9  * Input from keyboard to Navigate, or time based.
10 
11 ## Program Details :
12  * Written in C++.
13  * Opengl Library utilised
14  * Following functionality from Opengl implemented
15  * Keyboard Interactions.
16  * Mouse Interactions.
17  * 3D and 2D Transformations.
18  * Stroke Text.
19  * Timing Functions.
20  * Lighting and Shading.
21  * Perspective and Parallel Viewing.
22  * Texture Mapping.
23  * Soil Library required for image loading.
24  * Doxygen for Documentation Generation
25  * Generic Flipbook Class Created [ Library ]
26  - Book *flipbook = new flipbook(); -> To create a Flipbook at (0, 0, 0)
27  - flipbook->addPage(PAGE_TYPE_TEXT, char* string)
28  + string -> text here is rendered onto the screen at that Page.
29  - flipbook->addPage(PAGE_TYPE_DRAWING, funct)
30  + funct(xstart, ystart,z , xmax, ymax)
31  + Specified function called, and what is drawn in this function is rendered onto the page.
32  - flipbook->renderBook() -> Render Flipbook at that position.
33  - flipbook->renderPage() -> Render last page added.
34  - flipbook->renderPage(x) -> render Page x on screen.
